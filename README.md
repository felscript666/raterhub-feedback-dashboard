# Rating Feedback Dashboard

A web-based application for storing, organizing, and analyzing evaluation feedback records.

## Features

- Feedback management
- Rating statistics
- Search and filtering
- Performance tracking
- Responsive dashboard

## Technology Stack

- PHP
- MySQL
- JavaScript
- Bootstrap

## Installation

```bash
git clone https://github.com/felscript666/rating-feedback-dashboard.git
```

Configure the database connection and import the database schema before running the application.
